# Staque

A C++ implementation of a custom Staque (Stack + Queue) data structure using a doubly linked list.

In this implementation, odd numbers are inserted at the front of the structure while even numbers are inserted at the back.

## Features

- Doubly linked list implementation
- Deep copy constructor
- Assignment operator overloading
- Stream insertion operator overloading (<<)
- Unique value validation
- Search functionality
- Separate printing of odd and even elements
- Retrieval of the last inserted odd or even element
- Deletion of:
  - Even elements from right to left
  - Odd elements from right to left
  - Elements using LIFO order

## Technologies Used

- C++
- Dynamic Memory Management
- Doubly Linked Lists
- Object-Oriented Programming

## Project Structure

- Staque.h – Class declaration
- Staque.cpp – Class implementation
- driver.cpp – Test program

## Academic Purpose

This project was developed as part of a Data Structures course to demonstrate linked list manipulation, operator overloading, and memory management.
