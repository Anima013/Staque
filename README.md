# Staque

A C++ implementation of the Staque (Stack + Queue) data structure using a doubly linked list.

## Features

- Insert odd numbers at the front of the structure
- Insert even numbers at the end of the structure
- Delete elements according to specified conditions
- Search for values
- Display even and odd elements separately
- Copy constructor implementation
- Assignment operator overloading
- Stream insertion operator overloading
- Dynamic memory management

## Technologies Used

- C++
- Object-Oriented Programming
- Doubly Linked Lists

## Academic Purpose

This project was developed as part of a Data Structures course.
